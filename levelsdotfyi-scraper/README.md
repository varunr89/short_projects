# levelsdotfyi-scraper

Scraping salary, level, and experience data from levels.fyi using Selenium and BeautifulSoup.

## Data Source

- [levels.fyi](https://www.levels.fyi/comp.html?track=Product%20Manager) -- Product Manager compensation data

## Methodology

1. Launch headless Chrome via Selenium
2. Navigate to the PM compensation table and set 100 rows per page
3. Scrape level, experience, total compensation, and salary breakdown from each page
4. Clean and export to CSV
