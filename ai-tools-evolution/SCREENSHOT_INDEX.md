## Usage Notes

- **Best quality:** Use non-"_full.png" versions for standard viewport screenshots
- **Full page:** Use "_full.png" versions for complete page content
- **Mobile:** Use "v1_mobile_*" files for mobile documentation
- **Avoid duplicates:** "v1_screen_*" files are duplicates from first script run

## File Sizes

Most screenshots range from 31KB to 125KB, with GitHub screenshots being larger (97KB-471KB) due to more complex page content.
